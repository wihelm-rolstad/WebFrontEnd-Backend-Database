# WebFrontEnd-Backend-Database
Project to learn integration of databases to webapplications

This project is creating a fitness platform to track progression. <br>
This allows users to store lots of data such as userinformation, bodyweight, workouts, PRs/achievemnts.

This project is using the following tech-stack:

* Frontend: REACT JSX <br>
* BACKEND: JAVA (CONNECTED OVER REST API) <br>
* DATABASE: SUPABASE POSTGRESQL <br>

LIBRARIES: <br>
* REACT-CHARTJS-2 (for charts)
